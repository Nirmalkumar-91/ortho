# Event Website Skeleton (React)

This project contains a polished, single-page React skeleton inspired by a conference-style website layout.

All copy, labels, names, and visuals are dummy placeholders.
No original branding, trademarked naming, or external destination links are used.

## Sections Included

- Top bar and section navigation
- Hero area with CTA controls and key metrics
- About section and feature cards
- Faculty/speaker grid
- Three-day program timeline
- Workshop highlights
- Registration pricing cards
- Four-step registration process block
- Venue cards
- Contact cards
- Footer with placeholder policy actions

## Figma-Style Skeleton Spec

- Grid: responsive card-based blocks using repeated layout patterns
- Spacing rhythm: compact 8px-derived spacing units
- Visual language: rounded cards, soft outlines, warm neutral palette
- Typography: heading font `Sora`, body font `Manrope`
- Interaction style: clickable buttons only (no outbound URLs)
- Placeholder media: abstract gradient blocks and avatar/media placeholders
- Motion: subtle reveal and floating shape animation for hierarchy cues

## Run

```bash
npm install
npm run dev
```
